## Quick Setup Options

### Option 1: Using GitHub Web Interface (No Git Required - EASIEST)

1. **Go to GitHub** (https://github.com)
2. **Create new repository**:
   - Click "+" icon â†’ "New repository"
   - Name: `kidtools`
   - Make it Public
   - Check "Add a README file"
   - Click "Create repository"

3. **Upload files**:
   - Click "Add file" â†’ "Upload files"
   - Drag and drop `index.html` from `C:\kidtools\`
   - Click "Commit changes"

4. **Enable GitHub Pages**:
   - Go to repository Settings
   - Click "Pages" in left sidebar
   - Under "Source", select "main" branch
   - Click "Save"
   - Your site will be live at: `https://YOUR_USERNAME.github.io/kidtools/`

---

### Option 2: Using Git Command Line (If Git is installed)

1. **Install Git** (if not already installed):
   - Download from: https://git-scm.com/download/win
   - Install with default settings

2. **Run the deployment script**:
   - Open `C:\kidtools\`
   - Double-click `deploy-to-github.bat`
   - Follow the prompts
   - Enter your GitHub username when asked

3. **Enable GitHub Pages** (same as Option 1, step 4)

---

### Option 3: Using GitHub Desktop (User-Friendly GUI)

1. **Install GitHub Desktop**:
   - Download from: https://desktop.github.com/
   - Sign in with your GitHub account

2. **Create repository**:
   - File â†’ New Repository
   - Name: `kidtools`
   - Local Path: `C:\`
   - Click "Create Repository"

3. **Move files**:
   - Copy `index.html` and `README.md` to the repository folder

4. **Publish**:
   - Click "Publish repository"
   - Uncheck "Keep this code private"
   - Click "Publish Repository"

5. **Enable GitHub Pages** (same as Option 1, step 4)

## Troubleshooting

**Problem**: Site not loading after enabling Pages
**Solution**: Wait 2-3 minutes, then refresh. GitHub Pages can take a moment to build.

**Problem**: Firebase connection errors
**Solution**: Check your browser console (F12) for errors. Ensure Firebase config is correct.

**Problem**: Git not recognized
**Solution**: Install Git from https://git-scm.com/download/win
